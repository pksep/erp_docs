## Путь
Настройки (/settings) -> 6. Операции технологического процесса (/opertechproc)

---
## Описание
Страница "Операции технологического процесса" предназначена для управления операциями технологических процессов.\
Она позволяет пользователям добавлять новые операции, редактировать существующие или отправлять их в архив.\
Также на странице предусмотрена возможность назначения ответственных лиц за каждую операцию.\
Страница выполнена в виде таблицы, которая отображает список добавленных технологических операций и ключевые моменты по ним.

### Особенности
- **Интуитивный интерфейс**
    - Страница имеет простой и понятный интерфейс, что облегчает пользователям навигацию и управление операциями.
- **Табличное представление**
    - Операции представлены в виде таблицы, что позволяет легко просматривать и сравнивать информацию о различных операциях.
- **Назначение ответственных лиц**
    - Возможность назначения ответственных за каждую операцию, что улучшает управление и контроль за выполнением технологических процессов.

### Функционал
- **Просмотр операций**
    - Пользователи могут просматривать список всех добавленных технологических операций, включая ключевые моменты, такие как название операции, учет выработки сотрудником в норма часах, работа с чпу и ответственные лица.
- **Добавление новых операций**
    - Пользователи могут добавлять новые операции, заполняя необходимые поля информации.
- **Редактирование существующих операций**
    - Пользователи могут изменять информацию о существующих операциях, что позволяет поддерживать данные в актуальном состоянии.
- **Отправка в архив**
    - Пользователи могут отправлять ненужные операции в архив, что помогает поддерживать порядок в базе данных.
- **Назначение ответственных лиц**
    - Пользователи могут назначать ответственных за каждую операцию, что позволяет улучшить управление и контроль за выполнением процессов.

## Требования
- **Доступность**
    - Страница "Операции технологического процесса" должна быть доступна пользователям с соответствующими правами доступа (например, администраторам или менеджерам).
- **Интуитивный интерфейс**
    - Интерфейс страницы должен быть простым и понятным, обеспечивая легкость в использовании.
- **Таблица операций** - Должна отображать следующие столбцы:
    - **№**: Порядковый номер в таблице.
    - **Наименование операции**: Отображает наименование технологической операции.
    - **Учет выработки сотрудником в нормо-часах**:
       - **Подготовительное**: Чекбокс для учета подготовительных работ.
       - **Вспомогательное**: Чекбокс для учета вспомогательных работ.
       - **Основное**: Чекбокс для учета основных работ.
    - **С ЧПУ**: Чекбокс, указывающий, что операция выполняется с использованием ЧПУ.
    - **Ответственные лица**: 
       - Должен отображать количество ответственных лиц, прикрепленных к данной операции.
       - Выполнен в виде dropdown, показывающего общее количество прикрепленных сотрудников.
       - При наведении курсора должна появляться подсказка (tooltip) с табельным номером и инициалами ответственного сотрудника.
       - При клике должно открываться dropdown меню с выбором всех действующих сотрудников.
- **Компоненты управления**
   - Должны быть доступны три кнопки:
     - **Добавить**: Для добавления новой записи в таблицу.
     - **Редактировать**: Для изменения выделенной операции в таблице.
     - **Удалить**: Для удаления выделенной операции из таблицы.
- **Добавление новой записи**
   - При нажатии на кнопку "Добавить" должно открываться модальное окно для ввода информации о новой операции.
- **Редактирование записи**
   - При нажатии на кнопку "Редактировать" должно открываться модальное окно с текущими данными выделенной операции для редактирования.
- **Удаление записи**
   - При нажатии на кнопку "Удалить" должна появляться форма подтверждения, чтобы предотвратить случайные удаления.
- **Управление ответственными лицами**
   - Пользователь должен иметь возможность добавлять и удалять ответственных лиц через dropdown меню, которое открывается при клике на соответствующий элемент.
- **Чекбоксы для учета**
   - Чекбоксы для учета выработки должны корректно сохранять состояние и обновлять данные в системе.

## Описание таблицы "Операции технологического процесса" (Table 1)
Таблица "Операции технологического процесса" предназначена для отображения внесенных технологических операций и ключевых моментов по ним.\
Она предоставляет пользователям возможность управлять операциями и назначать ответственных лиц за каждую операцию.

### Структура
- **№**
    - Порядковый номер в таблице.
- **Наименование операции**
    - Отображает наименование технологической операции.
- **Учет выработки сотрудником в нормо-часах** - Разделено на три столбца и позволяет включить учет нормо-часов:
    - **Подготовительное**: Чекбокс для учета подготовительных работ.
    - **Вспомогательное**: Чекбокс для учета вспомогательных работ.
    - **Основное**: Чекбокс для учета основных работ.
- **С ЧПУ**
    - Чекбокс, указывающий, что операция выполняется с использованием станка ЧПУ (числовое программное управление).
- **Ответственные лица**
    - Отображает количество ответственных лиц, прикрепленных к данной операции.
    - Выполнен в виде dropdown, который показывает общее количество прикрепленных сотрудников.
    - При наведении курсора появляется подсказка (tooltip) с табельным номером, инициалами и должности ответственного сотрудника.
    - При клике открывается dropdown меню с выбором всех действующих сотрудников, включая информацию о табельном номере, инициалами и должностью.
    - В dropdown меню присутствует иконка крестик для удаления ответственных лиц.

### Особенности
- **Интуитивный интерфейс**
    - Таблица имеет простой и понятный интерфейс, что облегчает пользователям навигацию и управление операциями.
- **Чекбоксы для учета**
    - Учет выработки сотрудников осуществляется с помощью чекбоксов, что позволяет быстро и удобно фиксировать данные.
- **Динамическое управление ответственными лицами**
    - Возможность добавления и удаления ответственных лиц через dropdown меню упрощает управление операциями.

### Функционал
- **Добавление новой записи** (Button 1)
  - Кнопка "Добавить" позволяет пользователям добавлять новую запись в конце таблицы.
- **Редактирование записи** (Button 2)
  - Кнопка "Редактировать" позволяет изменять выделенную операцию в таблице, открывая соответствующее модальное окно для редактирования.
- **Удаление записи** (Button 3)
  - Кнопка "Удалить" удаляет выделенную операцию из таблицы, с подтверждением действия для предотвращения случайных удалений.
- **Управление ответственными лицами** (Dropdown 1)
  - Пользователи могут назначать ответственных лиц за каждую операцию, используя dropdown меню для выбора сотрудников и удаления их из списка ответственных.
- **Информационные подсказки**
  - Подсказки (tooltips) помогают пользователям быстро получать информацию о сотрудниках, ответственных за операции, что улучшает взаимодействие с таблицей.
