## Путь
Настройки (/settings) -> 5. Расходные материалы (типы и подтипы) (/materials/3)

---
## Описание
Страница "Расходные материалы (типы и подтипы)" предназначена для управления информацией о типах и подтипах расходных материалов в системе.\
Она предоставляет пользователям возможность просматривать существующие записи, а также создавать новые, редактировать или удалять уже имеющиеся.\
Это позволяет поддерживать актуальность данных о расходного материлах и их классификации.

### Особенности
- **Интуитивный интерфейс**
    - Страница имеет простой и понятный интерфейс, что облегчает пользователям навигацию и управление данными о расходных материалов.
- **Список типов и подтипов**
    - Пользователи могут видеть все типы и подтипы расходных материалов в одном месте, что упрощает их поиск и управление.
- **Функции фильтрации и поиска**
    - Возможность фильтрации и поиска по типам и подтипам расходных материалов для быстрого нахождения нужной информации.
- **Динамическая фильтрация**
    - При выборе типа в таблице, таблица с подтипами динамически фильтруется по выбранному типу и отображает только подтипы, относящиеся к этому типу.

### Функционал
- **Просмотр типов и подтипов**
    - Пользователи могут просматривать список всех типов и подтипов расходных материалов, включая их характеристики и описание.
- **Создание новых записей**
    - Пользователи могут добавлять новые типы и подтипы расходных материалов, заполняя необходимые поля информации.
- **Редактирование существующих записей**
    - Пользователи могут изменять информацию о типах и подтипах расходных материалов, что позволяет поддерживать данные в актуальном состоянии.
- **Удаление записей**
    - Пользователи могут удалять ненужные типы и подтипы расходных материалов, что помогает поддерживать порядок в базе данных.

## Требования
- **Доступность**
    - Страница "Расходные материалы (типы и подтипы)" должна быть доступна пользователям с соответствующими правами доступа.
- **Интуитивный интерфейс**
    - Интерфейс страницы должен быть простым и понятным, обеспечивая легкость в использовании.
- **Таблица "Тип (Тип профиля заготовки)"**
    - Должна отображать список всех типов расходных материалов с соответствующими столбцами.
    - Пользователи должны иметь возможность видеть все доступные типы.
- **Таблица "Подтип (Материал заготовки)"**
    - Должна отображать список подтипов расходных материалов, автоматически фильтруясь по выбранному типу из первой таблицы.
    - По умолчанию должна отображать список всех подтипов расходных материалов.
- **Компоненты управления**
    - Под каждой таблицей должны быть расположены три кнопки: "Создать", "Редактировать", "Удалить".
- **Навигация по таблицам**
    - Пользователь должен иметь возможность выбирать тип в первой таблице, что инициирует динамическое обновление второй таблицы с подтипами.
- **Просмотр типов и подтипов**
    - Пользователь должен иметь возможность просматривать список всех типов и подтипов расходных материалов, включая их характеристики и описание.
- **Создание новых записей**
    - При нажатии на кнопку "Создать" должно открываться соответствующее модальное окно для создания нового типа или подтипа расходного материла.
-**Редактирование существующих записей**
    - При нажатии на кнопку "Редактировать" должно открываться модальное окно для редактирования выбранного типа или подтипа расходного материла.
- **Удаление записей**
    - При нажатии на кнопку "Удалить" выбранная запись должна удаляться из соответствующей таблицы.

## Описание
На странице отображаются две таблицы: "Тип (Тип профиля заготовки)" и "Подтип (Материал заготовки)".\
Первая таблица содержит список типов расходных материалов, а вторая таблица отображает подтипы расходных материалов, которые автоматически фильтруются в зависимости от выбранного типа в первой таблице.\
По умолчанию в таблице подтипов отображается список всех подтипов расходных материалов.

### Структура
- **Таблица "Тип (Тип профиля заготовки)"** (Table 1)
  - Содержит столбцы с названиями типов расходных материалов.
  - Позволяет пользователям просматривать все доступные типы.
- **Таблица "Подтип (Материал заготовки)"** (Table 2)
  - Содержит столбцы с названиями подтипов расходных материалов.
  - Автоматически фильтруется по выбранному типу из первой таблицы.
  - По умолчанию отображает список всех подтипов расходных материалов.

### Особенности
- **Динамическая фильтрация**
    - Таблица подтипов автоматически обновляется в зависимости от выбранного типа в таблице типов, что упрощает поиск и управление данными.
- **Интуитивный интерфейс**
    - Простота и ясность интерфейса делают взаимодействие с таблицами легким и понятным для пользователей.
- **Управление записями**
    - Возможность создания, редактирования и удаления типов и подтипов расходных материалов через удобные кнопки управления.

### Функционал
- **Компоненты управления**
    - Под каждой таблицей расположены три кнопки:
- **Создание записей** (Button 1)
    - При нажатии на кнопку "Создать" открывается соответствующее модальное окно, где пользователи могут создать новый тип расходного материла или подтип расходного материла в зависимости от того, под какой таблицей была нажата кнопка.
- **Редактирование записей** (Button 2)
    - При нажатии на кнопку "Редактировать" открывается модальное окно, где пользователи могут редактировать выбранный тип расходного материла или подтип расходного материла, в зависимости от того, в какой таблице была выбрана запись.
- **Удаление записей** (Button 3)
    - При нажатии на кнопку "Удалить" выбранная запись удаляется из соответствующей таблицы, с подтверждением действия для предотвращения случайных удалений.
