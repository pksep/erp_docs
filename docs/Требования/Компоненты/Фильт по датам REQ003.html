Фильт по датам REQ003

Описание "Фильтр по датам REQ003"​

Компонент "Фильтр по датам" предназначен для фильтрации данных по дате или периоду.
Он позволяет пользователям выбирать начальную и конечную даты для отображения соответствующей информации.
Компонент выполнен в виде строки с метками и полями ввода для удобства использования.
Пример изображения фильтра по датам

Особенности​

  • Интуитивный интерфейс
    • Компонент имеет простой и понятный интерфейс, что облегчает процесс выбора дат.
  • Гибкость фильтрации
    • Возможность фильтрации как по отдельной дате, так и по диапазону дат, что позволяет пользователям точно настраивать отображаемую информацию.
  • Всплывающий календарь
    • При наведении курсора на поля ввода появляется всплывающий календарь, что упрощает выбор дат.
  • Кнопка сброса
    • Наличие кнопки "Сбросить период или дату" для быстрого возврата к исходным настройкам фильтрации.

Функционал​

  • Выбор периода
    • Поле ввода (Input 1) для выбора начальной даты с меткой "Выбрать период с:".
    • Поле ввода (Input 2) для выбора конечной даты с меткой "по:".
  • Сброс фильтра
    • Кнопка "Сбросить период или дату" очищает выбранные даты и возвращает компонент к исходному состоянию, позволяя пользователю начать новый выбор.
  • Динамическое обновление
    • При выборе дат компонент должен обновлять отображаемые данные в зависимости от заданного периода, обеспечивая актуальность информации.

Макет​

Элементы окнаНазначение
Input 1Поле ввода даты с
Input 2Поле ввода даты по
Button 1Сбрасывает значение фильтра по умолчанию для дат

Фильтр по датам​

  • Выбрать период, с: (Input 1)
    • Значение по умолчанию: сегодняшняя дата - 40 лет.
    • Формат: DD.MM.YYYY
    • Календарь: появляется при наведении курсора.
  • по: (Input 2)
    • Значение по умолчанию: сегодняшняя дата.
    • Формат: DD.MM.YYYY
    • Календарь: появляется при наведении курсора.
  • Сбросить период или дату (Button 1)
    • Функциональность: сбрасывает оба поля ввода на значения по умолчанию.
  • Условия
    • Оба поля должны содержать даты в формате DD.MM.YYYY.
    • Даты не могут быть пустыми.
