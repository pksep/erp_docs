## Описание страницы "Скомплектованные наборы"
Страница "Скомплектованные наборы" предназначена для управления скомплектованными наборами, предоставляя пользователям возможность разукомплектовать набор или передать его на сборку. 

### Особенности
- **Табличный интерфейс**
    - Страница выполнена в виде таблицы, которая отображает список скомплектованных наборов и информацию о них.
- **Сортировка**
    - Наборы отсортированы по дате планируемой отгрузки, начиная с самых последних.
- **Фильтрация**
    - **Фильтрация по наборам**: Позволяет фильтровать наборы по определенным критериям.
    - **Фильтрация по сборщикам**: Позволяет фильтровать наборы по сборщикам.
    - **Фильтр по заказам покупателей**: Позволяет фильтровать наборы по связанным заказам покупателей.
- **Сброс фильтров**
    - Кнопка для сброса всех примененных фильтров, возвращая таблицу к исходному состоянию.
- **Поиск**
    - Поиск по "Артикулу" и "Наименованию" набора для быстрого нахождения нужной информации.

### Функционал
- **Работа с наборами**
    - Пользователи могут разукомплектовать набор или передать его на сборку, что упрощает управление процессами.
    - При разукомплектации набора потомки возвращаются на остатки склада, и отнимается количество собранных наборов данной родительской сущности.
- **Интерактивные элементы**
    - Таблица содержит интерактивные элементы для удобного доступа к функциям управления наборами.
- **Удобный интерфейс**
    - Интуитивно понятный интерфейс, который позволяет пользователям легко ориентироваться и выполнять необходимые действия.

## Требования
- **Отображение данных**
   - Таблица должна отображать следующие колонки: *Иконка галочка*, *№*, *Изображение*, *Тип*, *Обозначение*, *Наименование*, *Заказано*, *Планируемая отгрузка*, *Надо собрать*, *Собрано*, *Информация о сборке набора* (с подколонками: *Кол-во*, *№ Заказа*, *№ комплектации*, *Сотрудник*, *Информация*, *Отгруженно по набору*), *Осталось собрать*, *На складе*, *Передано на сборку*, *Осталось передать*, *Информация об отгрузке*.
- **Интерактивные элементы**
   - Иконки должны быть четко обозначены и интуитивно понятны.
- **Сортировка**
   - Наборы должны быть отсортированы по дате планируемой отгрузки от самых последних.
- **Фильтрация по наборам**
   - Должен быть доступен фильтр с заголовком "Фильтрация по наборам:" и выпадающим меню для выбора статуса наборов (Все, Не переданные на производство, Не отгруженные полностью).
- **Фильтрация по сборщикам**
   - Должен быть доступен фильтр с заголовком "Фильтрация по сборщикам:" и кнопкой-иконкой для открытия модального окна "Выбрать сотрудника".
- **Сброс всех фильтров**
   - Должен быть доступен заголовок "Сбросить все фильтры:" с иконкой, при нажатии на которую отменяются все примененные фильтры.
- **Разукомплектовать наборы**
   - Должна быть кнопка "Разукомплектовать", позволяющая разукомплектовать отмеченные наборы, открывающая модальное окно "Набор для разукомплектации".
- **Передать на сборку**
   - Должна быть кнопка "Передать на сборку", позволяющая передать отмеченные наборы на сборку, открывающая модальное окно "Отметка о передачи набора на сборку".
- **Показать все наборы**
   - Должна быть кнопка "Показать все наборы", которая отображается только при переходе на страницу из накладной на комплектацию и позволяет отменить примененный фильтр.
- **Скорость загрузки**
   - Страница должна загружаться быстро, чтобы обеспечить комфортное взаимодействие пользователя с интерфейсом.
- **Обновление данных**
   - Данные в таблице должны обновляться в реальном времени при применении фильтров или выполнении операций.
- **Права доступа**
   - Доступ к странице и ее функционалу должен быть ограничен в зависимости от ролей пользователей.

## Описание таблицы "Скомплектованные наборы" (Table 1)
Таблица "Скомплектованные наборы" отображает все скомплектованные наборы и информацию о них, предоставляя пользователям возможность разукомплектовать набор или передать его на сборку.

### Структура
Таблица состоит из следующих столбцов:
- **[Иконка галочка](\img\IconCheckMark.png)**
    - Чекбокс для выбора наборов для дальнейших операций.
- **№**
    - Порядковый номер.
- **Изображение**
    - Изображение набора.
- **Тип**
    - Тип набора (Сб. или Изд.).
- **Обозначение**
    - Обозначение набора.
- **Наименование**
    - Наименование набора.
    - При клике открывается модальное окно "Краткая информация"
- **Заказано**
    - Количество заказанных наборов.
- **Поиск**
    - Расположен под описанными выше колонками и занимает всю их ширину.
    - Позволяет искать наборы по "Артикулу" или "Наименованию".
    - Включает возможность просмотра истории запросов поиска.
    - Используется placeholder для подсказки пользователю допустимых значений поиска.
- **Планируемая отгрузка**
    - Дата планируемой отгрузки.
- **Надо собрать**
    - Количество наборов, которые осталось скомплектовать для выполнения складских заказов, с учетом уже собранных наборов.
- **Собрано**
    - Количество скомплектованных наборов.
- **Информация о сборке набора**
    - Разделено на 6 подколонок, и каждая ячейка может быть разбита на несколько операций.
    - **Кол-во**: Количество скомплектованных наборов в одной операции.
    - **№ Заказа**: Номер заказа для скомплектованного набора.
    - **№ комплектации**: Порядковый номер комплектации в течение дня (формат: *цифра-текущая дата*).
    - **Сотрудник**: ФИО сотрудника, скомплектовавшего набор.
    - **Информация**: Дополнительная информация по скомплектованному набору.
    - **Отгруженно по набору**: Количество уже отгруженных наборов.
- **Осталось собрать**
    - Количество наборов, которые осталось скомплектовать для выполнения складских заказов, с учетом уже собранных наборов.
- **На складе**
    - Количество скомплектованных наборов.
- **Передано на сборку**
    - Количество переданных на сборку наборов.
- **Осталось передать**
    - Количество наборов, которые еще не переданы на сборку из уже скомплектованных, с учетом тех, что уже переданы на сборку.
- **Информация об отгрузке**
    - [Иконка](/img/plus.png), при клике на которую открывается модальное окно "Список отгрузок" с информацией об отгрузках.

### Особенности
- **Интерактивные элементы**
    - Чекбоксы и иконки для удобного выбора и доступа к дополнительной информации.
- **Сортировка**
    - Наборы отсортированы по дате планируемой отгрузки от самых последних.
- **Фильтрация**
    - Возможность фильтрации по различным критериям, что упрощает поиск нужных наборов.
- **Компоненты управления**
    - Фиксированное размещение элементов управления в нижней части экрана.

### Функционал
- **Разукомплектация** (Button 1)
    - Кнопка, позволяющая разукомплектовать отмеченные чекбоксом наборы. При нажатии открывается модальное окно "Набор для разукомплектации".
- **Передать на сборку** (Button 2)
    - Кнопка, позволяющая передать отмеченные чекбоксом наборы на сборку. При нажатии открывается модальное окно "Отметка о передачи набора на сборку".
- **Показать все наборы** (Button 5)
    - Кнопка, которая отображается только при переходе на страницу из накладной на комплектацию. Позволяет отменить примененный фильтр и показать все наборы в таблице.

## Описание фильтра "Фильтрация по наборам" (Dropdown)
Фильтр "Фильтрация по наборам" предназначен для удобного управления отображением данных в таблице "Скомплектованные наборы". Он позволяет пользователям быстро находить нужные наборы в зависимости от их статуса.

### Особенности
- **Интуитивный интерфейс**
    - Фильтр представлен в виде заголовка "Фильтрация по наборам:" с выпадающим меню (dropdown menu), что делает его использование простым и понятным.
- **По умолчанию установлено "Все"**
    - При загрузке страницы фильтр по умолчанию показывает все наборы, что позволяет пользователям видеть полную информацию.
- **Разнообразие опций**
    - Фильтр предлагает несколько вариантов для выбора, включая:
    - **"Все"**: Отображает все скомплектованные наборы.
    - **"Не переданные на производство"**: Фильтрует таблицу, показывая только наборы, которые еще не были переданы на производство.
    - **"Не отгруженные полностью"**: Фильтрует таблицу, отображая только наборы, которые не были полностью отгружены.

### Функционал
- **Фильтрация данных**
    - При выборе значения из выпадающего меню таблица "Скомплектованные наборы" автоматически обновляется, отображая только те наборы, которые соответствуют выбранному критерию.
- **Удобство использования**
    - Пользователи могут легко переключаться между различными фильтрами, что позволяет быстро находить нужные наборы и улучшает общую эффективность работы с таблицей.
- **Обновление в реальном времени**
    - Изменения в фильтре применяются мгновенно, что позволяет пользователям видеть результаты без необходимости перезагрузки страницы.

## Описание функции "Фильтрация по сборщикам" (Button 3)
Функция "Фильтрация по сборщикам" предназначена для управления отображением данных в таблице "Скомплектованные наборы" на основе выбранного сотрудника, ответственного за комплектацию наборов.

### Особенности
- **Интуитивный интерфейс**
    - Функция представлена с заголовком "Фильтрация по сборщикам:", что четко указывает на ее назначение.
- **[Иконка](/img/plus.png)**
    - Рядом с заголовком расположена [иконка](/img/plus.png), при нажатии на которую открывается модальное окно "Выбрать сотрудника".
- **Выбор сотрудника**
    - В модальном окне пользователи могут выбрать ФИО сотрудника, ответственного за скомплектованный набор.

### Функционал
- **Фильтрация данных**
    - После выбора сотрудника из модального окна таблица "Скомплектованные наборы" автоматически обновляется, отображая только те наборы, которые были скомплектованы выбранным сотрудником.
- **Удобство использования**
    - Пользователи могут быстро находить наборы, скомплектованные конкретным сотрудником, что упрощает управление и анализ данных.
- **Обновление в реальном времени**
    - Изменения в фильтре применяются мгновенно, позволяя пользователям видеть результаты без необходимости перезагрузки страницы.

## Описание функции "Сбросить все фильтры" (Button 4)
Функция "Сбросить все фильтры" предназначена для отмены всех примененных фильтров в таблице "Скомплектованные наборы", возвращая пользователей к полному списку наборов.

### Особенности
- **Ясный заголовок**
    - Функция представлена с заголовком "Сбросить все фильтры:", что четко указывает на ее назначение и действие.
- **Иконка для сброса**
    - Рядом с заголовком расположена [иконка](/img/plus.png), на которую можно кликнуть для выполнения сброса фильтров.

### Функционал
- **Отмена всех фильтров**
    - При нажатии на иконку все примененные фильтры к таблице "Скомплектованные наборы" отменяются, и таблица обновляется, отображая полный список наборов.
- **Удобство использования**
    - Пользователи могут быстро вернуть таблицу к исходному состоянию, что упрощает процесс работы с данными и позволяет легко начать новый поиск.
- **Интуитивно понятный интерфейс**
    - Функция легко доступна и понятна, что делает ее использование простым даже для новых пользователей.
