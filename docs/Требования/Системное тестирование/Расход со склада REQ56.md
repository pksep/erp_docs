## Описание
Страница "Расход со склада" предназначена для отображения информации об отгруженных изделиях по заказам покупателей.\
Она предоставляет пользователям возможность отслеживать все отгрузки, что позволяет эффективно контролировать выполнение заказов.

### Особенности
- **Интуитивный интерфейс**
    - Страница имеет простой и понятный интерфейс, что облегчает восприятие информации об отгрузках.
- **Фильтрация данных**
    - Пользователи могут фильтровать данные по датам или периодам, что позволяет быстро находить нужную информацию.
- **Печать содержания**
    - Возможность распечатать содержание таблицы для дальнейшего использования или отчетности.

### Функционал
- **Отображение информации об отгрузках**
    - Таблица показывает информацию о каждом изделии, включая данные о заказчике и номере заказа, по которому производилась отгрузка.
- **Фильтрация по датам или периодам**
    - Пользователи могут устанавливать диапазон дат для отображения информации об отгрузках за определенный период, что помогает в анализе данных.
- **Печать содержания таблицы**
    - Кнопка для отправки содержания таблицы на предпросмотр печати, позволяя пользователю распечатать информацию о расходах со склада.

## Требования
- **Заголовок страницы**
   - Заголовок должен отображать текст "Расход со склада".
- **Таблица "Расход со склада"**
   - Должна содержать две секции: "Накладные" и "Информация о заказе".
- **Столбцы таблицы в секции "Накладные"**
    - **№**: Порядковый номер в таблице.
    - **Дата**: Дата отгрузки.
    - **Основание**: Уникальный номер отгрузки.
- **Столбцы таблицы в секции "Информация о заказе"**
    - **№ Заказа**: Присвоенный номер заказа.
    - **Комплектация**: Иконка для доступа к информации о комплектации.
    - **Дата**: Дата создания заказа.
    - **Изделие**: Наименование заказанного изделия.
    - **Заказчик**: Название покупателя.
- **Отображение информации об отгрузках**
   - Таблица должна отображать актуальную информацию о расходах со склада, включая данные о накладных и заказах.
- **Фильтрация данных**
   - Пользователь должен иметь возможность фильтровать данные по датам или периодам для отображения информации за определенный период.
- **Поиск по Артикулу или Наименованию**
   - Должно быть поле для быстрого поиска по артикулу или наименованию изделий.
- **Доступ к модальным окнам**
   - При клике на ячейки в секции "Накладные" должно открываться модальное окно "Информация об Отгрузке".
   - При клике на ячейки в секции "Информация о заказе" должно открываться модальное окно "Заказ" с подробной информацией о заказе.
- **Печать содержания таблицы**
   - Кнопка "Печать" должна отправлять содержание таблицы на предпросмотр печати, позволяя пользователю распечатать информацию о расходах со склада.

## Описание таблицы "Расход со склада" (Table 1)
Таблица "Расход со склада" отображает информацию об отгруженных изделиях по заказам покупателей.\
Она разделена на две секции: "Накладные" и "Информация о заказе", что позволяет пользователям легко находить и просматривать данные о расходах и связанных заказах.

### Структура таблицы
- **Секция "Накладные"**
  - **№**: Порядковый номер в таблице.
  - **Дата**: Дата отгрузки.
  - **Основание**: Уникальный номер отгрузки.
    - При клике на ячейки данной секции открывается модальное окно "Информация об Отгрузке", где можно посмотреть подробную информацию об отгрузке данного заказа.
- **Секция "Информация о заказе"**
  - **№ Заказа**: Присвоенный номер заказа, по которому производился расход изделия со склада.
  - **Комплектация**: [Иконка](/img/plus.png), при клике на которую открывается модальное окно "Расход со склада", где отображается информация о комплектации изделия.
  - **Дата**: Дата создания заказа.
  - **Изделие**: Наименование заказанного изделия.
  - **Заказчик**: Название покупателя данного заказа.
    - При клике на ячейки данной секции открывается модальное окно "Заказ" выбранного заказа с подробной информацией о заказе.
- **Поиск**
  - Расположен под описанными выше колонками и занимает всю их ширину.
  - Позволяет искать детали по "Артикулу" или "Наименованию".
  - Включает возможность просмотра истории запросов поиска.
  - Используется placeholder для подсказки пользователю допустимых значений поиска.

### Особенности
- **Фильтр по дате или периоду**
    - Над таблицей расположен фильтр, который позволяет осуществить поиск по определенным критериям, что упрощает навигацию по данным.
- **Поиск по Артикулу или Наименованию**
    - Внутри таблицы под заголовками столбцов расположен поиск, который позволяет быстро находить нужные изделия.
- **Кнопка "Печать"**
    - Под таблицей расположена кнопка "Печать", при клике на которую содержание таблицы отправляется на предпросмотр печати.

### Функционал
- **Отображение информации об отгрузках**
    - Таблица показывает полную информацию о расходах со склада, включая данные о накладных и заказах.
- **Доступ к модальным окнам**
    - Пользователи могут открывать модальные окна для получения подробной информации об отгрузках и заказах.
- **Фильтрация и поиск**
    - Возможность фильтрации данных по дате и поиска по артикулу или наименованию для быстрого нахождения нужной информации.
- **Печать содержания таблицы** (Button 1)
    - Кнопка "Печать" позволяет пользователям распечатать информацию о расходах со склада для дальнейшего использования.
