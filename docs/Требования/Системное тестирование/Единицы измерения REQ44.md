## Путь
Настройки (/settings) -> Единицы измерения (/edizm)

## Описание
Страница "Единицы измерения" предназначена для хранения, редактирования, удаления и создания новых единиц измерения, которые будут использоваться в базе данных.\
Страница выполнена в виде таблицы, что позволяет удобно управлять различными категориями единиц измерения.

### Особенности
- **Удобный интерфейс**
    - Страница имеет интуитивно понятный интерфейс, который облегчает работу с единицами измерения.
- **Табличное представление**
    - Данные представлены в виде таблицы, что позволяет легко просматривать, сравнивать и управлять единицами измерения.
- **Категории**
    - Возможность группировки единиц измерения по различным категориям для упрощения поиска и управления.

### Функционал
- **Отображение единиц измерения**
  - Таблица отображает все внесенные единицы измерения с соответствующими категориями.
- **Создание новой единицы измерения**
  - Кнопка "Добавить" открывает модальное окно для ввода данных новой единицы измерения.
- **Редактирование единицы измерения**
  - Пользователь может выбрать существующую единицу измерения и отредактировать ее данные в модальном окне.
- **Удаление единицы измерения**
  - Возможность удаления единицы измерения с подтверждением действия для предотвращения случайных ошибок.

## Требования
- **Отображение данных** - Таблица должна отображать все существующие единицы измерения с колонками:
     - **Единица измерения**: название единицы измерения.
     - **Обозначение единицы**: краткое обозначение единицы.
- **Категоризация** - Таблица должна быть разбита на предустановленные категории:
    - Количественные единицы
    - Единицы массы
    - Единицы длины (Длина L, Ширина B, Высота H)
    - Единицы времени
    - Единицы площади
    - Единицы объема
    - Экономические единицы
    - Технические единицы
- **Поиск и фильтрация**
    - Пользователь должен иметь возможность фильтровать единицы измерения по категориям.
- **Сортировка**
    - Таблица должна поддерживать сортировку по колонкам "Единица измерения" и "Обозначение единицы".
- **Создание новой единицы измерения** - Кнопка "Создать" должна открывать модальное окно "Добавление единицы измерения", где пользователь может:
    - Выбрать категорию.
    - Ввести название единицы измерения.
    - Ввести обозначение единицы измерения.
    - При успешном добавлении должна отображаться уведомление об успешном создании.
- **Редактирование единицы измерения** - Кнопка "Редактировать" должна открывать модальное окно "Редактирование единицы измерения", где пользователь может изменить:
    - Название единицы измерения.
    - Обозначение единицы измерения.
    - При успешном редактировании должна отображаться уведомление об успешном обновлении.
- **Удаление единицы измерения**
    - Кнопка "Удалить" должна позволять удалить выбранную единицу измерения с подтверждением действия.
    - При успешном удалении должна отображаться уведомление об успешном удалении.

## Описание таблицы "Единиц измерений" (Table 1)
Таблица "Единиц измерений" отображает внесенные единицы измерения с возможностью создания новых, редактирования существующих или удаления.\
Таблица организована по предустановленным категориям, что упрощает навигацию и управление единицами измерения.

### Структура таблицы
- **Единица измерения**
    - Отображает название единицы измерения.
- **Обозначение единицы**
    - Отображает краткое обозначение единицы.
- **Категории** Таблица разбита на предустановленные категории:
    - Количественные единицы
    - Единицы массы
    - Единицы длины (Длина L, Ширина B, Высота H)
    - Единицы времени
    - Единицы площади
    - Единицы объема
    - Экономические единицы
    - Технические единицы

### Особенности
- **Интуитивный интерфейс**
    - Удобное представление данных в виде таблицы, что облегчает работу с единицами измерения.
- **Категоризация**
    - Разделение единиц измерения на предустановленные категории для упрощения поиска и управления.
- **Модальные окна**
    - Использование модальных окон для создания и редактирования единиц измерения, что улучшает пользовательский опыт.

### Функционал
- **Отображение единиц измерения**
    - Таблица показывает все внесенные единицы измерения с их названиями и обозначениями.

### Компоненты управления таблицей
- **Создать** (Button 1)
    - Кнопка "Создать" открывает модальное окно "Добавление единицы измерения", где пользователь может: "Выбрать категорию", "Ввести название единицы измерения", "Ввести обозначение единицы измерения".
- **Редактировать** (Button 2)
    - Кнопка "Редактировать" открывает модальное окно "Редактирование единицы измерения", где пользователь может изменить: "Название единицы измерения", "Обозначение единицы измерения".
- **Удаление** (Button 3)
    - Кнопка "Удалить" позволяет удалить выбранную единицу измерения с подтверждением действия для предотвращения случайных ошибок.
