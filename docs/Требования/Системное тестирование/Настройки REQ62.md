## Путь
Настройки (/settings)

---
## Описание
Страница "Настройки" выполняет роль навигации по ключевым настройкам программы.\
Она предоставляет пользователям удобный доступ к различным параметрам и конфигурациям системы.\
Страница выполнена в виде интерфейса с различными блоками, каждый из которых отвечает за определенные настройки.\
При клике на блок открывается соответствующая страница с детальными настройками.

### Особенности
- **Интуитивный интерфейс**
    - Страница имеет простой и понятный интерфейс, что облегчает пользователям навигацию по настройкам.
- **Блоки управления**
    - Каждый блок представляет собой отдельную категорию настроек, что позволяет пользователям быстро находить нужные параметры.
- **Динамическое обновление**
    - При выборе блока пользователи переходят на соответствующую страницу, где могут вносить изменения в настройки.

### Функционал
- **Навигация по блокам**
    - Пользователи могут кликать на различные блоки, чтобы открыть страницы с настройками, связанными с конкретной функцией или параметром системы.

## Описание
Страница "Настройки" служит центральным узлом для управления ключевыми параметрами программы.\
Она организована в виде блоков управления, каждый из которых предоставляет доступ к различным аспектам системы.\
При клике на любой блок пользователь переходит на соответствующую страницу, где может просматривать и изменять настройки.

### Особенности
- **Блоки управления**
    - Каждый блок представляет собой отдельную категорию настроек, что позволяет пользователям быстро находить нужные параметры.
- **Интуитивный интерфейс**
    - Простота и ясность интерфейса делают навигацию по настройкам легкой и понятной.
- **Темная и светлая темы**
    - Пользователи могут переключаться между светлой и темной темами интерфейса, что улучшает пользовательский опыт.

### Функционал
- **Список блоков управления**:
  1. **Сотрудники**: Переход на страницу `/employee` для просмотра и управления учетными записями пользователей.
  2. **Единицы измерения**: Переход на страницу `/edizm` для управления единицами измерения.
  3. **Базы материалов. Материалы для заготовок**: Переход на страницу `/materials/1` для работы с типами и подтипами материалов.
  4. **База материалов. Покупные детали**: Переход на страницу `/materials/2` для управления типами и подтипами покупных деталей.
  5. **База материалов. Расходные материалы**: Переход на страницу `/materials/3` для управления типами и подтипами расходных материалов.
  6. **Операции технологического процесса**: Переход на страницу `/opertechproc` для управления технологическими процессами.
  7. **База инструмента и оснастки**: Переход на страницу `/instrosn` для управления типами и подтипами инструментов и оснастки.
  8. **База оборудования**: Переход на страницу `/baseoborud` для управления типами и подтипами оборудования.
  9. **База непроизводственной техники и инвентаря**: Переход на страницу `/basetech` для управления типами и подтипами непроизводственной техники.
  10. **Дефицит на складе**: Переход на страницу `/diffsklad` для управления минимальными и рекомендуемыми остатками на складе.
  11. **Учет рабочего времени**: Переход на страницу `/uthetwtime` для просмотра информации об отработанном времени сотрудников.
  12. **Роли пользователей**: Переход на страницу `/rolesuser` для управления доступами по ролям.
  13. **Должности сотрудников**: Переход на страницу `/dolznuser` для работы с должностями сотрудников.
  14. **Разделы библиотеки**: Переход на страницу `/libbarysetting` для управления разделами библиотеки.
  15. **Стоимость норма-часа**: Переход на страницу `/pricehors` для управления стоимостью часовой ставки.
  16. **Сохранение баз данных, настроек, истории действий**: Переход на страницу `/savebasedata` для работы с резервными копиями и восстановлением данных.
  17. **История действий**: Переход на страницу `/actions` для просмотра истории действий пользователей.
- **Переключение тем**
    - Пользователи могут изменять тему интерфейса со светлой на темную и наоборот, что позволяет адаптировать внешний вид программы под свои предпочтения.
