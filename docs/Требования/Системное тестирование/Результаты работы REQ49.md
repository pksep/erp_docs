## Путь
Результаты работы (/resultwork)

## Описание
Страница "Результаты работы" предназначена для отображения подробной информации о истории действий сотрудников с отметками о выполнении.\
Она предоставляет пользователям возможность отслеживать и анализировать результаты работы, а также фильтровать данные по различным критериям.

### Особенности
- **Интуитивный интерфейс**
    - Страница имеет простой и понятный интерфейс, что облегчает восприятие информации о результатах работы.
- **Фильтрация данных**
    - Возможность фильтрации по сотруднику и дате или периоду, что позволяет пользователям быстро находить нужные записи.
- **Печать информации**
    - Возможность распечатать таблицу с результатами работы для удобства работы с документами.

### Функционал
- **Отображение истории действий**
    - Страница показывает подробную информацию о действиях сотрудников, включая отметки о выполнении, что позволяет пользователям быстро оценить результаты работы.
- **Фильтрация по сотруднику**
    - Пользователь может выбрать конкретного сотрудника для отображения его результатов работы.
- **Фильтрация по дате или периоду**
    - Пользователь может установить диапазон дат для отображения результатов работы за определенный период.
- **Печать таблицы**
    - Кнопка для отправки таблицы с результатами работы на предпросмотр печати, позволяя пользователю распечатать информацию.

## Требования
- **Доступность**
    - Страница должна быть доступна пользователям с соответствующими правами доступа.
- **Таблица "Результаты работы"** - Должна содержать следующие столбцы:
    - №
    - Табельный номер
    - Сотрудник
    - Дата
    - Наименование действий
    - Тип
    - Обозначение
    - Наименование
    - Кол-во, шт.
    - Примечание
- **Отображение данных**
   - Таблица должна отображать полную информацию о создании отметок о выполнении сотрудниками.
- **Сортировка**
   - Таблица должна сортироваться по дате проставления отметки, начиная с самых свежих.
- **Фильтрация**
   - Над таблицей должен быть доступен фильтр по датам, позволяющий пользователю находить отметки по заданным критериям.
- **Просмотр примечаний**
   - При клике на иконку примечания должно открываться модальное окно с дополнительной информацией.
- **Печать таблицы**
   - Кнопка "Печать" должна отправлять таблицу на предпросмотр печати, позволяя пользователю распечатать информацию.

## Описание функции "Фильтрации по сотрудникам:"
Функция фильтрации по сотрудникам представляет собой интерактивную [иконку](/img/plus.png), расположенную под заголовком страницы "Результаты работы".\
Эта иконка служит точкой входа для быстрой и наглядной фильтрации данных по конкретным сотрудникам, ответственным за создания отметки о выполнении.

### Особенности
- **Интуитивный интерфейс** 
    - [Иконка](/img/plus.png) фильтрации четко обозначена и легко доступна, что делает её использование простым и понятным для пользователей.
- **Модальное окно**
    - При активации [иконки](/img/plus.png) открывается модальное окно "Выбрать сотрудника", которое предоставляет пользователю список доступных сотрудников для выбора.
- **Выбор сотрудников**
    - В модальном окне пользователь может выбрать сотрудника который участвует в процессе создания отметки о выполнении, используя поиск по различным параметрам, таким как табельному номеру или инициалам. После выбора сотрудника таблица Результаты работы (Table 1) будет отфильтрована, отображая только те задачи, которые связаны с выбранными сотрудником.

### Функциональность
- **Применение фильтра**
    - После выбора сотрудников, таблица автоматически обновляется, отображая только те записи, которые связаны с выбранными сотрудниками.
- **Отмена фильтрации**
    - Пользователь имеет возможность отменить выбор или закрыть модальное окно без применения фильтрации.
- **Сбросить фильтр**
    - При нажатии кнопки "Сбросить фильтр" фильтрация по сотруднику отменяется, и таблица "Результаты работы" показывает все записи о выполнении.
- **Состояние**
    - Если фильтр по сотрудникам не был применен, кнопка остается скрытой и не отображается на странице.

## Описание
Таблица "Результаты работы" отображает подробную информацию о создании отметок о выполнении сотрудниками. Она предоставляет пользователям возможность отслеживать действия сотрудников и анализировать результаты их работы.

### Структура
- **№**
    - Порядковый номер в таблице.
- **Табельный номер**
    - Табельный номер сотрудника, создавшего отметку о выполнении.
- **Сотрудник**
    - Инициалы сотрудника, создавшего отметку о выполнении.
- **Дата**
    - Дата создания отметки.
- **Наименование действий**
    - Описание действий, при которых была проставлена отметка о выполнении (например, выполнение задания, выполнение операции с производства или сборки, действия в ПО Конструктор - создал, изменил и т.д.).
- **Тип**
    - Тип сущности, с которой работали при создании отметки.
- **Обозначение**
    - Обозначение сущности, в которой создавалась отметка.
- **Наименование**
    - Наименование сущности, в которой создавалась отметка.
- **Кол-во, шт.**
    - Количество штук, отмеченных при проставлении отметки о выполнении.
- **Примечание**
    - [Иконка](/img/plus.png), при клике открывается модальное окно "Примечание" и отображает примечание, оставленное при создании отметки.

### Особенности
- **Интуитивный интерфейс**
    - Удобное представление данных в виде таблицы, что облегчает работу с результатами.
- **Сортировка**
    - Таблица сортируется по дате проставления отметки, начиная с самых свежих.
- **Фильтрация**
    - Над таблицей расположен фильтр по датам, позволяющий находить отметки по необходимым критериям даты или периоду.
- **Печать информации**
    - Возможность распечатать таблицу с результатами работы для удобства работы с документами.

### Функционал
- **Отображение информации о результатах работы**
    - Таблица показывает все необходимые данные о создании отметок о выполнении, что позволяет пользователям быстро оценить состояние работы сотрудников.
- **Просмотр примечаний**
    - Пользователь может кликнуть на иконку примечания, чтобы получить детальную информацию о примечании, оставленном при создании отметки.
- **Фильтрация по датам**
    - Пользователь может установить диапазон дат для отображения результатов работы за определенный период.
- **Печать таблицы**
    - Кнопка "Печать" расположена внизу таблицы и отправляет таблицу на предпросмотр печати для последующей печати.

## Подключенные компоненты

### "Фильтр по датам REQ003"
Компонент "Фильтр по датам" предназначен для фильтрации данных по дате или периоду.\
Он позволяет пользователям выбирать начальную и конечную даты для отображения соответствующей информации.\
