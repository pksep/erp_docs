## Описание
Страница "Заказано у поставщиков" предназначена для создания заказов у наших поставщиков и производства.\
Она выполнена в виде таблицы, которая позволяет пользователям удобно просматривать заказы.\
Показывает лишь те сущности, для которых были созданы задачи на отгрузку.

### Особенности
- **Таблица с свитчерами**
    - Таблица содержит свитчеры для фильтрации типов заказов, что упрощает взаимодействие с данными.
- **Фильтрация по дате или периоду**
    - Пользователи могут фильтровать заказы по дате или периоду, что позволяет быстро находить нужные записи.
- **Краткая информация о заказе**
    - В таблице отображается краткая информация о каждом заказе, что помогает пользователям быстро оценивать статус и детали.
- **Кнопка "Создать заказ"**
    - Под таблицей расположена кнопка, которая позволяет пользователям выбрать поставщика и создать новый заказ.
- **Встроенный поиск**
    - Внутри таблицы имеется функция поиска по наименованию сущности или артикулу, что облегчает нахождение нужных заказов.
- **Модальное окно для подробностей заказа**
    - При клике на заказ открывается модальное окно, где можно просмотреть подробности выбранного заказа и отправить его в архив.

### Функционал
- **Создание заказов**
    - Пользователи могут создавать новые заказы у поставщиков или производства, выбирая необходимые параметры.
- **Фильтрация заказов**
    - Возможность переключения фильтрации существующих заказов через свитчеры в таблице.
- **Фильтрация и поиск**
    - Пользователи могут фильтровать заказы по дате и выполнять поиск по наименованию или артикулу для быстрого доступа к нужной информации.
- **Отображение информации**
    - Таблица предоставляет актуальную информацию о заказах, что помогает в планировании и управлении запасами.
- **Просмотр деталей заказа**
    - Пользователи могут просматривать подробности выбранного заказа в модальном окне и отправлять его в архив, что упрощает управление заказами.

## Требования
- **Заголовок страницы**
    - Заголовок должен отображать текст "Заказано у поставщиков".
- **Таблица "Заказано у поставщиков"** - Должна содержать следующие столбцы:
    - № Заказа
    - Дата создания
    - Наименование поставщика
    - № счета и дата
    - Сумма, руб.
    - Дата прихода
    - Статус
    - Подробнее (иконка с tooltip)
- **Создание заказов**
   - Пользователь должен иметь возможность создавать новые заказы у поставщиков или производства, выбирая необходимые параметры.
- **Фильтрация заказов**
   - Возможность фильтрации существующих заказов через свитчеры в таблице:
     - Все
     - От Поставщика
     - Только Сборки
     - Только Металлообработка
     - Архив
- **Поиск**
   - Должно быть поле для быстрого поиска по наименованию или артикулу, позволяющее находить нужные заказы.
- **Просмотр деталей заказа**
   - При клике на заказ должно открываться модальное окно с подробной информацией о выбранном заказе и возможностью отправить его в архив.
- **Получение информации о заказах**
   - Пользователь должен иметь возможность получать подробную информацию о каждом заказе через tooltip при наведении на иконку "Подробнее".

## Описание таблицы "Заказано у постащиков"
Таблица "Заказано у поставщиков" содержит заказы у производства и заказы у поставщиков. Она предоставляет пользователям возможность управлять заказами, отслеживать их статус и получать информацию о каждом заказе.

### Структура таблицы
- **№ Заказа**
    - Отображает номер созданного заказа.
- **Дата создания**
    - Дата создания заказа.
- **Наименование поставщика**
    - Наименование поставщика заказа.
- **№ счета и дата**
    - Отображает номер счета заказа и дату заказа.
- **Поиск**
    - Расположен под описанными выше колонками и занимает всю их ширину.
    - Позволяет искать детали по "Артикулу" или "Наименованию".
    - Включает возможность просмотра истории запросов поиска.
    - Используется placeholder для подсказки пользователю допустимых значений поиска.
- **Сумма, руб.**
    - Общая сумма заказа.
- **Дата прихода**
    - Планируемая дата прихода заказа.
- **Статус**
    - Отображает статус заказа.
- **Подробнее**
    - Иконка, при наведении на которую всплывает tooltip с информацией о заказе, включая:
        - Обозначение
        - Наименование
        - Единица измерения (ЕИ)
        - Количество
        - Сумма, руб. (за шт.)
        - Примечание

### Особенности
- **Сортировка по дате**
    - По умолчанию таблица сортируется по дате создания от самых свежих.
- **Поиск**
    - Под заголовками таблицы расположен поиск по наименованию или артикулу, что позволяет быстро находить нужные заказы.
- **Свитчеры для фильтрации** - Над таблицей расположены свитчеры для удобства фильтрации:
    - **Все**: Установлен по умолчанию и отображает все заказы.
    - **От Поставщика**: Фильтрует таблицу только по заказам у поставщика.
    - **Только Сборки**: Фильтрует таблицу и отображает только заказы сборки.
    - **Только Металлообработка**: Фильтрует таблицу и отображает только заказы металлообработки.
    - **Архив**: Фильтрует таблицу и отображает заказы, которые были перенесены в архив.

### Функционал
- **Управление заказами**
    - Пользователи могут просматривать, фильтровать и сортировать заказы по различным критериям.
- **Получение информации о заказах**
    - Возможность получения подробной информации о каждом заказе через tooltip при наведении на иконку "Подробнее".
- **Создание нового заказа**
    - Под таблицей расположена кнопка "Создать заказ", при клике на которую открывается модальное окно с возможностью выбора, у кого будет производиться заказ (Поставщик, Деталь, Сборка, Изделие).
- **Информация о заказе**
    - При нажатии на любую ячейку заказа открывается модальное окно "Заказ", содержащее подробную информацию о выбранном заказе.
