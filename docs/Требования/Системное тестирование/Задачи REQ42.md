## Описание страницы "Задачи"
Страница "Задачи" предназначена для управления задачами, распределения их по сотрудникам и создания различных проектов и досок.\
Она обеспечивает пользователям удобный интерфейс для организации работы и эффективного взаимодействия.

### Особенности
- **Многоуровневая структура**
    - В каждом проекте можно создавать несколько досок, а в каждой доске — колонки с различными названиями для группировки задач.
- **Гибкость управления**
    - Пользователи могут легко переключаться между проектами, досками и задачами, что упрощает организацию рабочего процесса.
- **Интуитивно понятный интерфейс**
    - Удобное взаимодействие с элементами управления, включая Drag and Drop для перемещения задач.

### Функционал
- **Создание проектов**
    - Возможность создавать новые проекты с уникальными названиями и описаниями.
- **Создание досок**
    - В рамках каждого проекта пользователи могут добавлять доски для организации задач.
- **Создание колонок**
    - Каждая доска может содержать колонки с различными названиями для упрощения группировки задач.
- **Создание задач**
    - В каждой колонке можно создавать множество задач, каждая из которых включает:
    - **Заголовок задачи**: Заголовок задачи.
    - **Иконки исполнителей**: Отображают привязанные иконки ответственных за выполнение задачи.
    - **Иконка статуса выполнения**: Пустой круг для невыполненных задач и заполненный зеленым для выполненных.
    - **Теги**: Отображаются при наличии, для быстрой идентификации задач.
    - **Иконки комментариев**: Если есть комментарии, отображается иконка сообщения, при наведении курсора на которую показывается количество комментариев. Если комментариев нет, иконка скрыта.
    - **Иконка скрепки**: При наведении курсора отображается количество прикрепленных файлов. Если прикрепленных файлов нет, иконка скрыта.
    - **Иконка календаря**: При наведении курсора на задачу появляется иконка календаря, при нажатии на которую можно установить дату для задачи. Если дата установлена, она отображается.
    - **Подзадачи**: Имя подзадачи отображается, если они есть.
    - **Время выполнения**: В задаче может отображаться временная шкала. При назначении времени выполнения задачи или при её запуске отображается количество затраченного времени. Также можно остановить или продолжить выполнение задачи.
    - **Приоритет**: При установке приоритета блок с задачей обводится рамкой соответствующего цвета, а фон задачи изменяется.
    - **Номер задачи**: Отображает хэштег номера задачи.
- **Перемещение задач**
    - Все проекты, доски и задачи можно перемещать с помощью функции Drag and Drop, что позволяет легко реорганизовывать рабочие процессы.
