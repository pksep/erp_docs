Выбрать сотрудника REQ8

Путь​

Производство (/production) -> Металлообработка (/metalloworking) -> Клик на иконка Фильтрация по сборщикам:*

Описание​

Модальное окно "Выбрать сотрудника" предназначено для выбора сотрудника из таблицы со списком сотрудников.
Пользователи могут также осуществлять поиск сотрудника по инициалам или табельному номеру, что упрощает процесс нахождения нужного сотрудника.

Особенности​

  • Интуитивный интерфейс
    • Модальное окно имеет простой и понятный интерфейс, что облегчает выбор сотрудника.
  • Поиск
    • Возможность поиска по инициалам и табельному номеру для быстрого нахождения нужного сотрудника.
  • Таблица сотрудников
    • Отображение списка сотрудников в виде таблицы, что позволяет легко просматривать доступные варианты.

Функционал​

  • Выбор сотрудника
    • Пользователь может выбрать сотрудника из таблицы, кликнув на соответствующую строку.
  • Поиск
    • Поле для ввода инициалов или табельного номера, позволяющее фильтровать список сотрудников.
  • Закрытие модального окна
    • Возможность закрыть модальное окно без выбора сотрудника, возвращая пользователя к предыдущему интерфейсу.
  • Подтверждение выбора
    • Кнопка "Выбрать" для подтверждения выбора сотрудника, которая активируется только при выборе сотрудника из списка.

Требования​

  • Открытие окна
    • Модальное окно должно открываться по нажатию на соответствующую кнопку на основной странице.
  • Информация о пользователях
    • В верхней части окна должно отображаться общее количество активных пользователей в формате: "Активные пользователи: Количество активных пользователей".
    • При применении параметров поиска Количество активных пользователей изменяется в соответствии с применённым фильтром и отображает количество найденных сотрудников.
  • Структура таблицы
    • Таблица должна содержать три столбца:
      • Табельный номер: отображает уникальный номер сотрудника.
      • Должность: отображает текущую должность сотрудника.
      • Инициалы: отображает ФИО сотрудника.
  • Поиск сотрудников
    • Поле поиска должно находиться под заголовками столбцов и содержать placeholder с подсказкой о допустимых значениях (по инициалам или табельному номеру).
    • Пользователь должен иметь возможность вводить данные в поле поиска для фильтрации списка сотрудников.
    • Результаты поиска должны обновляться в реальном времени, отображая только тех сотрудников, которые соответствуют введенным критериям.
  • Выбор сотрудника
    • Пользователь должен иметь возможность выбрать сотрудника, кликнув на соответствующую строку таблицы.
    • Кнопка "Выбрать" должна быть расположена под таблицей и активироваться только при выборе сотрудника.
    • При нажатии на кнопку "Выбрать" модальное окно должно закрыться с выбранным сотрудником
  • Передача данных
    • При нажатии на кнопку "Выбрать" выбранный сотрудник должен передаваться на основную страницу для дальнейшего взаимодействия.
  • Закрытие окна
    • Модальное окно должно закрываться при клике вне области модального окна.
    • При закрытии все введенные данные и выбранные сотрудники должны сбрасываться.
  • Уведомления
    • При нажатии на кнопку "Выбрать", если ни один сотрудник не выбран, должно появляться предупреждение: "Сначала выберите сотрудника"

Описание таблицы "Выбрать сотрудника" (Table 1)​

Таблица "Выбрать сотрудника" отображает список всех зарегистрированных пользователей в системе.
В верхней части таблицы указано общее количество активных пользователей.
Таблица позволяет пользователям легко находить и выбирать нужного сотрудника для дальнейшего взаимодействия.

Структура таблицы​

  • Табельный номер
    • Отображает табельный номер сотрудника.
  • Должность
    • Отображает должность сотрудника.
  • Инициалы
    • Отображает ФИО сотрудника.
  • Поиск
    • Расположен под описанными выше колонками и занимает всю их ширину.
    • Позволяет искать сотрудника по "Инициалам" или "Табельному номеру".
    • Включает возможность просмотра истории запросов поиска.
    • Используется placeholder для подсказки пользователю допустимых значений поиска.

Особенности​

  • Общее количество активных пользователей
    • Строка с текстом "Активные пользователи: Количество активных пользователей" располагает над таблицей.
  • Поиск
    • Поле поиска позволяет пользователям быстро находить сотрудников по инициалам или табельному номеру.
  • Сортировка
    • Таблица сортируется по табельному номеру, что упрощает навигацию по списку сотрудников.

Функционал​

  • Отображение списка сотрудников
    • Пользователи могут видеть всех зарегистрированных сотрудников с их табельными номерами, должностями и инициалами.
  • Поиск
    • Пользователь вводит данные в поле поиска, и таблица обновляется, отображая только тех сотрудников, которые соответствуют введенным критериям.
  • Выбор сотрудника (Button 1)
    • Кнопка "Выбрать" расположена под таблицей для выбора отмеченного сотрудника.
    • Нажатие на кнопку "Выбрать" позволяет пользователю выбрать отмеченного сотрудника для дальнейшего взаимодействия с основной страницей.
  • Обновление количества пользователей
    • Общее количество активных пользователей обновляется в реальном времени в зависимости от параметров поиска.
