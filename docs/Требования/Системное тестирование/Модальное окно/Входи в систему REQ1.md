## Путь

Входи в систему (http://dev.npoamotiv.ru/)

---

## Описание

Страница авторизации предназначена для входа пользователя в систему и защиты от несанкционированного доступа. Она обеспечивает безопасный доступ к функционалу системы, позволяя только авторизованным пользователям выполнять действия в приложении. Страница выполнена в виде блока с данными и блока с цифрами, что упрощает процесс ввода информации.

## Особенности

- **Интуитивный интерфейс**: Страница имеет простой и понятный интерфейс, что облегчает пользователям процесс авторизации.
- **Динамическое обновление полей**: При выборе табельного номера автоматически обновляется логин пользователя, а при выборе логина — табельный номер, что минимизирует вероятность ошибок.
- **Использование цифрового блока**: Правый блок с цифрами позволяет пользователям вводить данные с помощью кнопок, что удобно для ввода пароля и табельного номера.

## Функционал

- **Блок "Вход в систему"**:

  - **Dropdown menu "Табельный номер"**: Позволяет выбрать табельный номер сотрудника. При выборе автоматически обновляется поле "Логин" с привязанным логином пользователя.
  - **Dropdown menu "Логин"**: Отображает инициалы сотрудника. При выборе автоматически обновляется поле "Табельный номер" с привязанным табельным номером.
  - **Поле ввода "Пароль"**: Позволяет вводить пароль. Пользователи могут вводить пароль как с клавиатуры, так и с помощью правого блока с цифрами.

- **Кнопка "Войти"**:

  - При нажатии на кнопку происходит проверка введенной информации на валидацию. Если данные корректны, пользователь получает доступ к системе.

- **Подсказки**:

  - Под кнопкой "Войти" расположена подсказка, которая изменяется в зависимости от этапа ввода:
    - Первоначально: "Начните вводить табельный номер".
    - После ввода табельного номера: "Начните вводить пароль".

- **Правый блок с цифрами**:

  - Содержит кнопки с цифрами от 0 до 9, символ стрелки, направленный влево (для удаления последнего введенного символа), и кнопку "ВВОД", которая инициирует процесс валидации введенных данных.

- **Перенаправление**:
  - При успешной проверке данных пользователя перенаправляют на стартовую страницу системы, предоставляя доступ к функционалу приложения.
