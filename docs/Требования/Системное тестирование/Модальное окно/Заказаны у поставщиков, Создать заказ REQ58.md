## Путь
Склад (/sclad) -> Заказано у поставщиков (/ordersuppliers) -> Кнопка "Создать заказ"

---
## Описание
Модальное окно открывается при нажатии на кнопку "Создать заказ" на странице "Заказано у поставщиков".\
Оно позволяет пользователям выбрать, у кого будет совершен заказ, предоставляя удобный интерфейс для выбора типа заказа.

### Особенности
- **Кнопки выбора** - Модальное окно выполнено в виде четырех кнопок с названиями:
    - **Поставщик**
    - **Детали**
    - **Сборка**
    - **Изделие**
- **Интуитивный интерфейс**: Простота и ясность интерфейса позволяют пользователям быстро ориентироваться и делать выбор.

### Функционал
- **Выбор типа заказа**
    - При нажатии на одну из кнопок ("Поставщик", "Детали", "Сборка", "Изделие") открывается модальное окно "Создание заказа поставщика".
- **Создание заказа**
    - В новом модальном окне пользователи могут выбрать соответствующий тип сущности и запустить его в производство или заказать у поставщика.

---
## Требования
- **Интуитивный интерфейс**
    - Интерфейс модального окна должен быть простым и понятным, обеспечивая легкость в использовании.
- **Функциональность кнопок**
    - Каждая кнопка должна корректно открывать соответствующее модальное окно для создания заказа.
- **Проверка выбора**
    - Пользователь должен иметь возможность выбрать только одну из кнопок для создания заказа, и при этом должно открываться модальное окно с выбранным типом поставщика.
