## Путь
Задачи на отгрузку (/issueshipment) -> Иконка в столбце "Комплектация/особенности заказа"

---
## Описание
Модальное окно "Комплектация заказа *Номер заказа*" предназначено для отображения потомков первого порядка, необходимых для комплектации родительской сущности по указанному заказу.\
Оно выполнено в виде таблицы, которая предоставляет пользователям информацию о каждом потомке, необходимом для завершения комплектации.

### Структура таблицы
- **№**
    - Порядковый номер.
- **Обозначение**
    - Обозначение потомка.
- **Наименование**
    - Наименование потомка.
- **Кол-во**
    - Количество потомков, необходимых для комплектации общего количества родительской сущности по заказу.

### Особенности
- **Интуитивный интерфейс**
    - Модальное окно имеет простой и понятный интерфейс, что облегчает пользователям навигацию и взаимодействие с данными о комплектации.
- **Табличное представление**
    - Информация о потомках представлена в виде таблицы, что позволяет легко просматривать и анализировать данные.
- **Кнопка "Печать"**
    - Возможность отправки комплектации на предпросмотр печати, что упрощает процесс документирования.

### Функционал
- **Кнопка "Печать"**
    - При нажатии на кнопку "Печать" информация о комплектации отправляется на предпросмотр печати, позволяя пользователям проверить данные перед печатью.
