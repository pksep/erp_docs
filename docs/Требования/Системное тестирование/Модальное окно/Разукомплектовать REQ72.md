## Путь
Склад (/sclad) -> Скомплектованные наборы (/complect_kit/0) -> Нажатие на кнопку "Разукомплектовать"

---
## Описание
Модальное окно "Разкомплектовать" предназначено для отображения наборов сущностей, которые были отмечены на странице "Скомплектованные наборы". Оно открывается при нажатии на кнопку "Разукомплектовать". В данном окне пользователи могут просмотреть скомплектованные наборы каждой отмеченной сущности и имеют возможность разукомплектовать всю партию или частично.

### Особенности
- **Интуитивный интерфейс**
    - Модальное окно имеет простой и понятный интерфейс, что облегчает пользователям навигацию и взаимодействие с набором сущностей.
- **Отображение скомплектованных наборов**
    - В окне представлены все скомплектованные наборы, что позволяет пользователям легко просматривать информацию о каждой сущности.
- **Гибкость разукомплектования**
    - Пользователи могут выбрать, разукомплектовать ли всю партию или только определенные элементы, что обеспечивает гибкость в управлении запасами.

### Функционал
- **Просмотр скомплектованных наборов**
    - Пользователи могут видеть список всех скомплектованных наборов, связанных с отмеченными сущностями, включая детали о каждом наборе.
- **Выбор разукомплектования**
    - **Полное разукомплектование**: Возможность разукомплектовать всю партию скомплектованного набора.
    - **Частичное разукомплектование**: Возможность выбрать конкретные элементы для разукомплектования, что позволяет пользователям управлять запасами более эффективно.

## Требования
- **Доступность**
    - Модальное окно "Разкомплектовать" должно быть доступно пользователям с соответствующими правами доступа.
- **Интуитивный интерфейс**
    - Интерфейс модального окна должен быть простым и понятным, обеспечивая легкость в использовании.
- **Блок сущности**:
    - Должен отображать название блока в формате "Набор для разукомплектовки № *порядковый номер*".
    - Должен отображать наименование и обозначение сущности.
- **Таблица "Скомплектованные наборы"** - Должна содержать следующие столбцы:
    - **№**: Порядковый номер.
    - **№ Набора**: Номер скомплектованного набора.
    - **Кол-во в наборе**: Количество скомплектованных наборов операции.
    - **Передано на сборку**: Количество переданных наборов на сборку.
    - **Разукомплектовать**: Поля ввода для изменения количества наборов, которые будут разукомплектованы.
- **Подсказка**:
    - Должна находиться под таблицей и информировать пользователей о том, что наборы не разукомплектуются, если в столбце "Разукомплектовать" значение будет равно нулю.
- **Кнопка "Разукомплектовать"**:
    - Должна быть доступна для инициации процесса разукомплектования.
- **Просмотр информации о сущности**
    - Пользователь должен иметь возможность видеть название, обозначение и информацию о скомплектованных наборах для каждой отмеченной сущности.
 **Изменение количества для разукомплектования**
    - Пользователь должен иметь возможность вводить числовые значения в поле "Разукомплектовать", чтобы указать, сколько наборов он хочет разукомплектовать.
- **Инициация разукомплектования**
    - При нажатии на кнопку "Разукомплектовать" система должна обрабатывать запрос и разукомплектовывать только те наборы, для которых в столбце "Разукомплектовать" указано значение больше нуля.
- **Валидация ввода**
    - Поля ввода для разукомплектования должны принимать только числовые значения. Если введено недопустимое значение, должно отображаться сообщение об ошибке.
- **Закрытие модального окна**
    - Пользователь должен иметь возможность закрыть модальное окно без применения изменений.

## Описание
Набор для разукомплектовки отображает отмеченную сущность в форме блока, содержащего информацию о наборе и таблицу скомплектованных наборов.\
Пользователи могут изменять количество наборов для разукомплектования и инициировать процесс разукомплектования.

### Заголовок
- **Название блока**
    - "Набор для разукомплектовки № *порядковый номер*".
- **Наименование**
    - Отображает наименование сущности.
- **Обозначение**
    - Обозначение сущности.

### Структура таблицы
- **№**
    - Порядковый номер.
- **№ Набора**
    - Номер скомплектованного набора.
- **Кол-во в наборе**
    - Количество скомплектованных наборов в партии.
- **Передано на сборку**
    - Количество переданных наборов на сборку из партии.
- **Разукомплектовать**
    - Поля ввода, где можно изменить количество наборов, которые будут разукомплектованы (по умолчанию значение соответствует количеству в операции). Допустимые значения — только положительные числовые.

### Особенности
- **Интуитивный интерфейс**
    - Модальное окно имеет простой и понятный интерфейс, что облегчает пользователям навигацию и взаимодействие с набором сущностей.
- **Гибкость разукомплектования**
    - Пользователи могут изменять количество наборов для разукомплектования, что обеспечивает гибкость в управлении запасами.
- **Подсказка о разукомплектовании**
    - Информативная подсказка помогает пользователям понять, как работает процесс разукомплектования.

### Функционал
- **Просмотр информации о сущности**
    - Пользователи могут видеть название, обозначение и информацию о скомплектованных наборах для каждой отмеченной сущности.
- **Изменение количества для разукомплектования**
    - Пользователи могут вводить числовые значения в поле "Разукомплектовать", чтобы указать, сколько наборов они хотят разукомплектовать.
- **Инициация разукомплектования**
    - При нажатии на кнопку "Разукомплектовать" система обрабатывает запрос и разукомплектовывает только те наборы, для которых в столбце "Разукомплектовать" указано значение больше нуля.
- **Валидация ввода**
    - Поля ввода для разукомплектования должны принимать только числовые значения, что предотвращает ошибки ввода.
- **Кнопка "Разукомплектовать"**
    - При нажатии разукомплектовывает все наборы сущности, где в столбце "Разукомплектовать" значение не равно нулю.
- **Уведомление**
    - При успешной разукомплектации набора всплывает уведомление об успешности операции.
- **Закрытие модального окна**
    - При клике вне области модального окна.
    - При успешной разукомплектации.
